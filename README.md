# Native Base

A static Next.js site for **Native Base**, powered by [alfabolt](https://alfabolt.com).

## Colors

- **Primary:** `#1a1a1a`
- **Accent / CTA:** `#fe0534`
- **White:** `#ffffff`

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Static export

```bash
npm run build
```

Output is in the `out/` folder. Deploy that folder to any static host.
